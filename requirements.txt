# =============================================================================
# Post-installation notes:
#
# 1. Tesseract OCR binary (required for pytesseract):
#    brew install tesseract
#
# 2. Ghostscript (required for camelot PDF rendering):
#    brew install ghostscript
#
# 3. Ollama (required for local LLM serving):
#    brew install ollama
#    OR: curl https://ollama.ai/install.sh | sh
#
# 4. MLX training (Mac only — skip on Linux):
#    mlx-lm is already in requirements.
#    On Linux/CUDA, replace with: pip install unsloth[colab-new]
#
# 5. After training, create Ollama model:
#    ollama create camels-analyst -f ./Modelfile
#    ollama serve   (in a separate terminal)
